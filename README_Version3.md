# Ambient Stars — Demo

简短说明：一个可交互的氛围生成器（星光 + 背景合成器）。用于作品集展示与交互演示。

运行：
1. 下载或克隆代码
2. 用浏览器直接打开 index.html（或部署到 GitHub Pages）
3. 点击 Start Sound 以启用背景音（浏览器需要用户交互来启动音频）

技术栈：HTML/CSS/JavaScript、p5.js、Tone.js

产品亮点：
- 视觉与音频联动，用户可调节色调与强度
- 低门槛、易部署，适合做 30s demo 视频

扩展想法：
